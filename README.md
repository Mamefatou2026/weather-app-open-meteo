# Weather App

## Démo

L'application est disponible ici : https://github.com/Mamefatou2026/weather-app-open-meteo

## Features

- Utilisation de l'API Open-Meteo pour récupérer les données météo
- Localisation pré-configurée via un fichier `config.json` (plus de moteur de recherche)
- Affichage de l’heure et de la date locales actuelles
- Température, humidité et code météo
- Vitesse et direction du vent
- Heures de lever et de coucher du soleil
- Système métrique ou impérial sélectionnable
- Rafraîchissement automatique des données toutes les heures
- Gestion des erreurs et écran de chargement

## Installation

```bash
# Cloner le projet
git clone https://github.com/Mamefatou2026/weather-app-open-meteo.git

# Se déplacer dans le dossier du projet
cd weather-app-open-meteo

# Installer les dépendances
npm install

# (Optionnel) Créer un fichier .env.local si nécessaire
# cp .env.example .env.local

# Lancer l'application en développement sur un port disponible
npm run dev
```
